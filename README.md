#University project 
<h3>Initial commands to run:</h3>
<ul>
  <li>
    <h4>Front-end directory</h4>
    <ul>npm i</ul>
    <ul>npm start</ul>
  </li>
  <li>
    <h4>BFF/Middleware (Back-end for Front-end) directory</h4>

  <li>
    <h4>Back-end directory</h4>

  </li>
</ul>
